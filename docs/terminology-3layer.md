# 3-Layer Terminology System (3층 용어 시스템)

## 규칙
사주 용어가 본문에 **처음 등장**할 때 3-Layer 전체를 표기한다.
이후 반복 등장 시에는 Layer 1 (영어)만 사용한다.

### Layer 구조
| Layer | 설명 | 예시 |
|-------|------|------|
| Layer 1 | 자연스러운 영어 (Natural English) | Eating God |
| Layer 2 | 고전 번역 / 로마자 (Classical + Romanization) | *Siksin* |
| Layer 3 | 한자 + 한글 (Hanja + Korean) | (식신 食神) |

### 첫 등장 표기법
> The **Eating God** — known as *Siksin* (식신 食神) — represents creative output energy in your chart.
