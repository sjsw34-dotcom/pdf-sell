## 11. 주의사항

### 중복 콘텐츠 절대 금지
- 16권 + Master Edition + 워크북 = 총 18개 상품
- **어떤 두 상품 사이에서도 같은 문장이 나오면 안 됨**
- 같은 "甲木 Yang Wood"를 18번 다르게 써야 함
- Claude Code 작성 시 매번 기존 상품들과 크로스체크 필수

### Amazon 콘텐츠 정책
- "excessively reused, recycled, or repeated content"는 KDP에서 거부됨
- 16권이 구조가 비슷하더라도 **내용은 각각 독립적**이어야 함
- Draft2Digital도 동일 정책: "low-content" 또는 "recycled content" 거부

### 시리즈 D 주의
- 5개 상생 조합만 다룸 (Wood→Fire, Fire→Earth, Earth→Metal, Metal→Water, Water→Wood)
- 상극 조합(Wood→Earth 등)도 만들 수 있지만 → "클래시 시리즈"로 별도 확장 가능
- 1차 출시에서는 상생 5조합만. 반응 보고 확장 결정

### 커버 디자인 일관성
- 16권 모두 "SajuMuse" 브랜드 아이덴티티 통일
- 시리즈별 색상 코드:
  - A 시리즈: 각 오행 색상 (Wood=Green, Fire=Red, Earth=Yellow, Metal=White, Water=Blue)
  - B 시리즈: 주제별 아이콘 (Love=Heart, Money=Coin, Health=Lotus, Year=Horse)
  - C 시리즈: 해당 년도 동물 (2026=Horse, 2027=Goat...)
