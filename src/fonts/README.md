# Fonts

Place the following `.ttf` files in this directory before building PDFs.

## English Body — Inter
- `Inter-Regular.ttf`
- `Inter-Bold.ttf`

Download: https://fonts.google.com/specimen/Inter

## English Title — Playfair Display
- `PlayfairDisplay-Regular.ttf`
- `PlayfairDisplay-Bold.ttf`

Download: https://fonts.google.com/specimen/Playfair+Display

## CJK Fallback — Noto Sans KR
- `NotoSansKR-Regular.ttf`

Used for rendering Chinese characters (한자) in saju charts.
Download: https://fonts.google.com/noto/specimen/Noto+Sans+KR

> **Note**: These files are not committed to git due to size.
> Add `src/fonts/*.ttf` to `.gitignore`.
